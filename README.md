WABot Convert Image to PDF

## Run Locally

Clone the project

```bash
  git clone https://github.com/fahmiibrahimdevs/wabot-cv-image-to-pdf
```

Go to the project directory

```bash
  cd wabot-cv-image-to-pdf
```

Install dependencies

```bash
  npm install
```

Install dependencies **recommended**
```bash
 npm install --force
```

Start the server

```bash
  npm start
```
